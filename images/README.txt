IMAGES FOLDER — Luck Is For Dudes
=================================

Put your own photos and artwork in this folder.

Suggested file names (match these in index.html if you use them):
  - about.jpg     → photo for the About section
  - product-tee.jpg
  - product-mug.jpg
  - product-keyring.jpg
  - product-print.jpg

Suggested sizes (rough guide):
  - About photo: about 800 x 600 pixels (or similar landscape shape)
  - Product photos: about 600 x 600 pixels (square works well)

Supported formats: .jpg, .jpeg, .png, .webp

Tips:
  - Keep file names simple: no spaces if you can help it (use-dashes-like-this.jpg)
  - After adding a photo, open index.html and update the image path if needed
  - Big files (over 2MB) can make the page slow — compress them if needed
    (sites like tinypng.com can help)

The website works without custom photos — it uses built-in comic-style
placeholders until you swap them in.
